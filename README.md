# 🎓 Student Management System (Python)

A simple **Student Management System** built using Python.  
This project performs basic student record management using a menu-driven program.

---

## 🚀 Features

- ➕ Add Student
- 📄 View Students
- 🔍 Search Student
- ✏️ Update Student
- ❌ Delete Student
- 💾 Data stored in memory / JSON (optional upgrade)

---

## 🛠️ Technologies Used

- Python 3
- VS Code
